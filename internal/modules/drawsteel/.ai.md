# Draw Steel Module

## Purpose

Provides Draw Steel RPG reference content by MCDM Productions. Serves
ability descriptions, creature stat blocks, and ancestry features as
tooltips (on @mention hover) and dedicated reference pages.

## Tier

Module (Game System -- optional, enabled per campaign)

## Domain Concepts

- **CC-BY Content:** Creative Commons Attribution content from MCDM's SRD.
  We can include CC-BY-4.0 licensed content with attribution.
- **Tooltip:** A small HTML fragment shown on hover when a user @mentions a
  game reference (e.g., `@ability:telekinetic-strike` shows keywords, damage).
- **Reference Page:** A full page showing complete details for an ability,
  creature, ancestry, etc.

## Files

| File | Purpose |
|------|---------|
| manifest.json | Module metadata and category definitions |
| handler.go | Serves reference pages and tooltip API |
| data/ | Static JSON files with CC-BY content |
| templates/ | Reference page and tooltip Templ components |

## Dependencies

- **Uses:** Nothing (read-only, self-contained)
- **Used by:** widgets/mentions (tooltip API), entity pages (reference links)

## Business Rules

- **Read-only.** This module NEVER writes to the database.
- Only CC-BY-4.0 licensed content with MCDM attribution.
- Tooltip API returns HTML fragments for hover previews.
- Data files are loaded once at startup and cached in memory.
